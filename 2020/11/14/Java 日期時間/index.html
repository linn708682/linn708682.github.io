時間格式

時間的表示主要用SimpleDateFormat來設計，有許多的格式字母可運用來設計出所需的日期樣式，以下介紹幾個常用的，詳細可以參考JAVA API: https://docs.oracle.com/javase/7/docs/api/java/text/SimpleDateFormat.html 或是 https://www.runoob.com/java/java-date-time.html

  • G:西元
  • y:年
  • M:月
  • d:日
  • h:小時(12進制)
  • H:小時(24進制)
  • m:分
  • s:秒
  • S:毫秒
  • E:星期幾
  • a:AM/PM標記 結果:
