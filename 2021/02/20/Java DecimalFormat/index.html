Java DecimalFormat

簡介

DecimalFormat可以用來將我們所要表達的數值調整至我們所需要的樣式，是常會使用到的工具之一。

說明

可以使用DecimalFormat來
1.取得所需的位數(用#及0)
2.可加入其他所需的字符
3.可處理金錢表示中的千位分隔符號
以上的應用都可以混著使用
